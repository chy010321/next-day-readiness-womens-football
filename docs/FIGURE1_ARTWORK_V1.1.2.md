# Figure 1 artwork - v1.1.2

## Canonical manuscript asset

`figures/liuchengtu.png` is the canonical Figure 1 artwork for v1.1.2. It replaces the compact scripted study-design schematic used in v1.1.1. The artwork is a static publication figure and is intentionally retained as a versioned visual asset; it is not regenerated by the numerical analysis code.

## Methods-to-graphic mapping

The figure shows, from left to right:

1. SoccerMon subjective monitoring inputs.
2. Paired complete-case eligibility and next-calendar-day target alignment.
3. Source-team 2020-only preprocessing, chronological tuning, and comparator-specific population-model development.
4. P0 (current readiness only) and P1 (current readiness plus load, wellness, calendar, and readiness-reporting features).
5. Separate P0 and P1 past-only, model-specific residual personalisation, including up to 28 earlier eligible forecast dates for the same player and a cold-start option.
6. Within-team temporal evaluation and bidirectional cross-team transportability under sequential target-team adaptation.
7. Incremental-value assessment using Delta MAE, percentile player-cluster bootstrap confidence intervals, and secondary diagnostics.

## Interpretation boundary

The figure is a methods schematic. It contains no performance values, no sample-size claims, and no causal interpretation. It does not describe zero-shot external validation: target-team prior outcomes are available for sequential residual updating.

## Technical note

The canonical `liuchengtu.png` is a 3688 x 1706 px PNG tagged at 600 dpi. It was produced by twofold Lanczos resampling of the approved 1844 x 853 px source artwork, preserved in `figures/source_artwork/`; this improves raster-delivery compatibility but does not convert the image into native vector artwork. For journal submission, prefer a vector redraw or an editorially supplied native high-resolution export when available. The present source is intended for the full-width manuscript placement used in `main.tex`; do not downscale it below the manuscript's current Figure 1 width without checking legibility.
