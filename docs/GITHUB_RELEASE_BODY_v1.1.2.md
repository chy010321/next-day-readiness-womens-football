# GitHub Release body - v1.1.2

## v1.1.2 - Figure 1 artwork and reproducibility revision

This release provides the reproducibility package for:

> *Incremental Value of Daily Workload and Wellness Monitoring Beyond a Personalised Readiness-History Baseline for Next-Calendar-Day Readiness in Elite Women's Football*

### What changed

- Replaced the manuscript Figure 1 with a content-rich methodological pipeline (`figures/liuchengtu.png`).
- Updated the Figure 1 caption and manuscript source to align the visual workflow with the calendar-day forecasting protocol, source-team 2020 development, P0/P1 comparator definition, separate model-specific past-only residual personalisation, four evaluation settings, and the incremental-value estimand.
- Added versioned Figure 1 artwork documentation and v1.1.2 release/post-archive DOI instructions.
- Updated README, CFF citation metadata, Zenodo metadata, manifests, and release documentation to v1.1.2.

### Effect on reported findings

This release does not introduce a new numerical analysis. The selected penalties, primary estimates, and wellness-item-missingness sensitivity estimates are unchanged from the audited v1.1.1 analysis.

### Data availability

The original SoccerMon source data are not redistributed. This release contains code, manuscript source, figures, tables, documentation, and non-identifying aggregate reference outputs.
