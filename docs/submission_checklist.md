# Pre-submission checklist

## Authorship and declarations
- [ ] Replace the bracketed author, affiliation, correspondence, funding, and CRediT placeholders.
- [ ] Confirm all authors meet the target journal's authorship criteria.
- [ ] Confirm competing-interest wording.
- [ ] Obtain or document a local ethics determination for secondary analysis of public, de-identified data if your institution requires it.

## Reproducibility
- [ ] Deposit the derived player-day analysis data, code, and package/environment information in Zenodo, OSF, or GitHub + Zenodo.
- [ ] Add the permanent DOI to the Data availability and Code availability sections.
- [ ] Verify that all reported results reproduce from the final locked analysis script.

## Reporting and submission
- [ ] Check the target journal's current author instructions, word count, reference style, tables/figures limit, and required reporting checklist.
- [ ] Check all citations and DOIs against publisher records.
- [ ] Confirm the title, abstract, tables, figure legends, and supplementary material match the final analysis outputs.
- [ ] Run a final language edit and plagiarism/similarity check.
- [ ] Ensure all rights to use/reproduce figures and data are satisfied.
