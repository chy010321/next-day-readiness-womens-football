# Remaining action before formal journal submission

## DOI for code and reproducibility materials

Create a permanent Zenodo DOI for the final code package before formal journal submission. Follow `docs/DOI_DEPOSIT_GUIDE.md`.

After Zenodo creates the DOI, update this line in `sections/05_declarations.tex`:

```tex
\url{https://doi.org/[INSERT-ZENODO-DOI]}
```

Replace the bracketed placeholder with the **version DOI** for the exact archived release, for example:

```tex
\url{https://doi.org/10.5281/zenodo.12345678}
```

## Final author verification

Before submission, personally verify the statistical results, figures, tables, citations, authorship statements, data/code declarations, and the final target journal's current instructions.
