# Direct-upload Overleaf project: next-day readiness forecasting (v5)

This LaTeX package can be uploaded directly to Overleaf and compiled with **pdfLaTeX + Biber**.

## Upload to Overleaf

1. In Overleaf, select **New Project** -> **Upload Project**.
2. Upload the ZIP archive without extracting it.
3. Confirm `main.tex` is the main document.
4. Compile with **pdfLaTeX** and **Biber**.
